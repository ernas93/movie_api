MOVIE API

This movie API will allow users to sign up, create a list of their favourite movies, change their personal information, as well as accessing general information of different movies.

Endpoints documentation

Endpoint Description URL HTTP Method Query Parameters Request body format Response data format
Greeting endpoint / GET - - A text message containing a greeting message
Login a user /login POST - - A JSON object allowing an existing user to login
Returns all movies /movies GET - - A JSON object holding data about all the movies
Returns movie by title and the information about that movie /movies/:title GET :title - A JSON object holding data about a single movie
Returns data about a genre by genre name /movies/genre/:genreName GET :genreName - A JSON object holding data about a specific movie genre
Returns data about a director by name /directors/:directorName GET :directorName - A JSON object holding data about a specific director
Allows new users to register /users POST - A JSON object holding data about the new user to add, structured like: { "name" : "Tim" } A JSON object holding data about the user that was added
Allows users to update users information /users/:Username PUT :Username A JSON object holding data about the users information A JSON object holding data about the updated user information
Allows users to add a movie to their list of favourite movies /users/:Username/:MovieId PUT :Username, :MovieId - A JSON object holding data about the user and the movie that was added
Allows users to remove a movie from their list of favourite movies /users/:Username/:MovieId DELETE :Username, :MovieId - A JSON object holding data about the user and the movie that was removed
Allows existing users to deregister /users/:Username DELETE :Username - A text message indicating whether the user has successfully deregistered
